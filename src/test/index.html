The effort.less grid system allows you to set your own grid parameters. By default effort.less is configured for 12 columns, 768 pixel total width, and 3% gutters. These settings can be overridden by simply setting them in your own .less files:


                @import "../effort.less";

                @grid-columns: 12;
                @grid-width: 768;
                @grid-gutter: .03;

                // ... your own styles here

effort.less allows you to set up multiple grid systems on a single page. This gives you fine grain control over your layouts when you need it. Simply override the @grid-columns, @grid-size, or @grid-gutter parameter to adjust your grid. Any element that is a direct child and has a class of "column" will be considered to be a single column within the row. Any element within the row can be marked with column specificity e.g.: 

                <div class="column four">Some Content</div>>
It is also possible to keep the grid semantics outside the markup by specifying what is to be considered a single column. 

                .myRowClass
                {
                    // Title area we want as 4 column rather than 12
                    @grid-columns: 4;

                    // this class as a row, this will prepare the grid
                    .row;

                    // Then mark that we want any div tag that is a direct child to be a column
                    > div
                    {
                        .column;
                    }

                    // And for fun, any span would be considered 2 columns
                    > span
                    {
                        .column(2);
                    }
                }
